# Sistema de Cadastro em Java

## Sobre

Projeto desenvolvido durante o curso Técnico em Desenvolvimento de Sistemas com o objetivo de praticar os conceitos fundamentais da linguagem Java.

O sistema implementa operações CRUD (Create, Read, Update e Delete), permitindo o gerenciamento de registros e reforçando conceitos de Programação Orientada a Objetos.

## Tecnologias

* Java
* Programação Orientada a Objetos

## Funcionalidades

* Cadastro de registros
* Consulta
* Atualização
* Exclusão

## Aprendizados

* Classes e Objetos
* Encapsulamento
* Organização de código
* Estruturas de decisão
* Estruturas de repetição
* CRUD em Java
